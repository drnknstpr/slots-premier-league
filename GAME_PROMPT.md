# Промпт для генерации игры: Slots & Premier League

## Концепция игры
Slot + Rogue игра с футбольной тематикой (Английская Премьер-лига). Игрок выбирает клуб АПЛ и с помощью слот-машины должен победить остальные 19 команд лиги.

## Технические требования

### Стек технологий
- **Frontend**: HTML5, CSS3, JavaScript (Vanilla или React/Vue)
- **Стилизация**: Современный UI с футбольной тематикой
- **Анимации**: Плавные анимации спинов, попаданий, побед
- **Данные**: Статические данные по командам и игрокам АПЛ

### Структура игры

#### 1. Экран выбора клуба
- **Заголовок**: "ЭТО СЛОТЫ АПЛ! ПОЙМАЙ УДАЧУ И РАЗВАЛИ ОСТАЛЬНЫЕ 19 КЛУБОВ АПЛ"
- **Интерфейс**: Сетка из 20 клубов АПЛ с логотипами/названиями
- **Доступность**: Все клубы видны, но активен только "Арсенал" (остальные затемнены с замком/индикатором "скоро")
- **Данные клубов**:
  - Арсенал (игрок)
  - Бернли (первый оппонент, HP: 1200)
  - Вулверхэмптон (HP: 1500)
  - ... и так далее для всех 20 клубов

#### 2. Игровой экран (слот-машина)

**Компоненты:**
- **Слот-машина**: 3 слота, вертикальная прокрутка
- **Информация о противнике**: 
  - Логотип/название команды
  - HP бар (текущее/максимальное)
  - Прогресс: "X/19 команд побеждено"
- **Спины**: Счетчик "5 спинов в этом раунде"
- **Игроки в слотах**: 
  - Фото игрока (AI-генерированное местоholder или реальное фото)
  - Рейтинг (число)
  - Редкость (цветовая индикация: обычный/редкий/эпический/легендарный)

**Механика спинов:**
- При нажатии на рычаг/кнопку "SPIN" запускается анимация прокрутки слотов
- Каждый слот останавливается с задержкой (визуально приятно)
- После остановки всех слотов показываются 3 игрока Арсенала
- Суммируется рейтинг игроков = урон по оппоненту
- Урон вычитается из HP оппонента с анимацией
- Уменьшается счетчик спинов

**Условия победы/поражения:**
- Если HP оппонента <= 0 за 5 спинов → победа, переход к следующему противнику
- Если HP оппонента > 0 после 5 спинов → поражение, игра начинается заново

#### 3. Экран между раундами (Rogue-механика)

После победы над командой:
- Показываются 3 случайных игрока из Арсенала
- Игрок выбирает 1 игрока для усиления
- Усиленный игрок получает:
  - x2 к силе (рейтинг)
  - x2 к частоте дропа в слотах (больше шанс выпадения)
- Анимация "усиления" с визуальным эффектом

#### 4. Игровые данные

**Команды АПЛ (20 клубов) с HP:**
```
Арсенал: игрок
Бернли: 1200 HP
Вулверхэмптон: 1500 HP
... (остальные команды с постепенно увеличивающимся HP)
```

**Игроки Арсенала с рейтингами и редкостью:**
- **Обычные (Common)**: 40-60 рейтинг (Saka, Martinelli, Raya)
- **Редкие (Rare)**: 70-90 рейтинг
- **Эпические (Epic)**: 100-120 рейтинг
- **Легендарные (Legendary)**: 130+ рейтинг

Примерный список игроков Арсенала:
- Bukayo Saka: 100 (Epic)
- Gabriel Martinelli: 70 (Rare)
- David Raya: 60 (Common)
- Martin Ødegaard: 95 (Epic)
- Declan Rice: 90 (Rare)
- ... и другие

#### 5. Визуальный стиль

- **Цветовая схема**: Зеленый футбольный газон, контрастные цвета команд
- **Типографика**: Жирные, читаемые шрифты
- **Анимации**: 
  - Плавная прокрутка слотов
  - Вспышки при нанесении урона
  - Празднование при победе
  - Плавные переходы между экранами
- **UI элементы**: Крупные кнопки, четкая визуальная иерархия

#### 6. Игровая логика

**Рандомизация выпадения игроков:**
- Вероятность выпадения зависит от редкости
- Усиленные игроки имеют x2 шанс выпадения
- Балансировка: чтобы игра была интересной, но не слишком легкой/сложной

**Прогрессия сложности:**
- Каждый следующий противник сильнее предыдущего
- HP увеличивается постепенно
- Игрок должен стратегически усиливать игроков

**Состояние игры:**
- Не сохраняется (как указано в требованиях)
- При перезагрузке начинается заново
- Все данные в памяти во время игры

2. **Производительность**: Плавные 60 FPS анимации
3. **Фидбек**: Четкая обратная связь на все действия игрока

**Раунд 1:**
