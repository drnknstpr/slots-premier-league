# Slots & Premier League

Прототип слот-игры с rogue-механикой и футбольной тематикой (Английская Премьер-лига).

## Описание

Игра в жанре Slot + Rogue, где игрок выбирает клуб АПЛ (Арсенал) и с помощью слот-машины должен победить остальные 19 команд лиги.

## Механика игры

1. **Выбор клуба**: Выберите Арсенал (в прототипе доступен только он)
2. **Слот-машина**: 3 слота с игроками Арсенала
3. **Урон**: Сумма рейтингов выпавших игроков = урон по оппоненту
4. **Раунды**: 5 спинов на каждого оппонента
5. **Усиления**: Между раундами выбирайте игрока для усиления (x2 к силе и частоте выпадения)

## Установка и запуск

1. Откройте `index.html` в браузере
2. Или используйте локальный сервер:
   ```bash
   python -m http.server 8000
   ```
   Затем откройте http://localhost:8000

## Структура проекта

- `index.html` - Основной HTML файл
- `style.css` - Стили игры
- `game.js` - Игровая логика
- `data.js` - Данные команд и игроков
- `GAME_PROMPT.md` - Детальный промпт для разработки
